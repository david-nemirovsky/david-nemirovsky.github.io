Some Cool Plotly Plots

Data Import, Sampling, and Tidying

First, let’s get the restaurant data:

Now, let’s take a sample of 10,000 restaurant inspections in NYC:

set.seed(6)
sample_df = 
  nyc_inspections %>% 
  sample_n(10000)

Time for some data tidying:

sample_df = 
  sample_df %>% 
  mutate(
    borough = as.factor(boro),
    critical_flag = as.factor(critical_flag),
    grade = as.factor(grade),
    score = as.double(score),
    grade_date = as.Date(grade_date)) %>% 
    select(dba, borough, cuisine_description, critical_flag, score, grade, grade_date) %>% 
    filter(grade == c("A", "B", "C"))

The resulting sample dataset is composed of 1620 observations.

Plot 1

Now, let’s make a plot to see the distribution of grades for Chinese food restaurants throughout the 5 boroughs from our sample:

sample_df %>% 
  filter(str_detect(cuisine_description, "[Cc][Hh][Ii][Nn][Ee][Ss][Ee]")) %>% 
  group_by(borough, grade) %>% 
  count(grade) %>% 
  plot_ly(x = ~grade, y = ~n, color = ~borough, type = "bar", colors = "viridis") %>% 
  layout(title = "Distribution of Chinese-Food Inspection Grades", xaxis = list(title = "Grade"), yaxis = list(title = "Count"))

Plot 2

This time, let’s plot the mean inspection scores for all types of restaurants in the 5 boroughs within our sample (note that higher scores result from more hazardous health violations):

sample_df %>% 
  group_by(borough, score) %>% 
  summarize(mean_score = mean(score)) %>% 
  plot_ly(x = ~borough, y = ~mean_score, type = "box", color = ~borough, colors = "viridis") %>% 
  layout(title = "Mean Inspection Scores for All Restauarnt Types", xaxis = list(title = "Borough"), yaxis = list(title = "Mean Score"))

Plot 3

Finally, let’s plot inspection scores for Mexican food restaurants over time in our sample.

sample_df %>%
  filter(str_detect(cuisine_description, "[Mm][Ee][Xx][Ii][Cc][Aa][Nn]")) %>% 
  plot_ly(x = ~grade_date, y = ~score, type = "scatter") %>% 
  layout(title = "Inspection Scores for Mexican-Food Restaurants Over Time", xaxis = list(title = "Borough"), yaxis = list(title = "Score"))
